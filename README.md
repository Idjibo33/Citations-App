# Citation App
Ce projet est une mini application d'affichage de citation provenant d'un d'un api externe.

## Fonctionnalité
- **Affichage de citations populaires et leurs auteurs**


## Objectif du projet: 
L'objectif est de créer une application mobile de lecture de citation de motivation permettant à l'utilisateur de lire des citations.

## Technologies utilisées: 
- **Framework:** Flutter
- **Language:**Dart
- **UI:** Pageview...
- **Packages:** Dio, provider
